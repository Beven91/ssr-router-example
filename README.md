# game-2048


### 一、 简介

本用例游戏代码部分来源于[react-native-2048](https://github.com/liuhong1happy/react-native-2048)

### 二、 使用

**web**

- 安装

        npm install react-native-on-web-cli -g

- 启动

        react-native-on-web start

**ios**

- 安装

        npm install

- 启动

        react-native run-ios

**android**

- 安装

        npm install

- 启动

        react-native run-android
